# AlcoholFree
### Intro
1. 투입 인원 : Front-End 2명 / Back-End 4명
2. 개발 기간 : 2023-10-23 ~ 2023-11-03(2주)
3. 프로젝트
  - 프로젝트 명 : alcoholfree
  - 프로젝트 목적 : 나날이 성장 하고 있는 주류 시장은 온라인에서의 판매가 제한적이기 때문에, 판매가 아닌 판매를 중계하는 사이트를 만들어 보고자 기획하게 되었음

  - 이재현 : 주문
